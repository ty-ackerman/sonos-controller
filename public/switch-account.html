Switch Sonos Account

To switch to a different Sonos account, you'll need to log in using an incognito/private window to bypass cached sessions.

  1. Click the button below to open the login page
  2. Right-click the new tab that opens
  3. Select "Open in Incognito Window" (Chrome) or "Open Link in New Private Window" (Firefox)
  4. Complete the login in the incognito window
  5. Return to this tab and click "I've Completed Login"
